# 🚌 Умный Городской Транспорт Алматы (Smart Transit)

## Как запустить этот проект (ИНСТРУКЦИЯ ДЛЯ ЧАЙНИКОВ)

Если вы вообще не шарите в программировании, просто делайте всё шаг за шагом:

### Шаг 1: Установите нужные программы
Вам понадобятся всего две программы на вашем компьютере:
1. **Node.js** (это движок, на котором всё работает). 
   - Скачать можно тут: [https://nodejs.org/](https://nodejs.org/) (скачивайте ту версию, где написано "LTS" или "Рекомендуется для большинства").
   - Просто установите её как обычную игру, везде нажимая "Далее".
2. **Git** (чтобы легко скачать код с интернета).
   - Скачать тут: [https://git-scm.com/](https://git-scm.com/) (также просто везде "Далее").

---

### Шаг 2: Скачайте этот код себе на компьютер
1. Откройте любую папку на компьютере (например, на Рабочем столе).
2. Нажмите правой кнопкой мыши по пустому месту в папке и выберите **"Open Git Bash here"** (Открыть Git Bash здесь).
3. Откроется черное окно. Вставьте туда эту команду (чтобы вставить, нажмите правую кнопку мыши -> Paste) и нажмите Enter:
   ```bash
   git clone https://github.com/Jasya228/bus-ssys.git
   ```

---

### Шаг 3: Запуск СЕРВЕРА (Мозги системы)
1. Откройте папку `bus-ssys`, которая только что скачалась.
2. Зайдите внутрь папки `backend`.
3. В самом верху папки (где написан путь, например `C:\Users\Desktop\bus-ssys\backend`) кликните левой кнопкой мыши, сотрите весь текст, напишите слово `cmd` и нажмите Enter. Откроется черное окошко.
4. Введите там команду установки библиотек:
   ```bash
   npm install
   ```
   *Подождите, пока скачаются файлы (побегут строчки).*
5. Затем запустите сервер командой:
   ```bash
   npm run dev
   ```
   **Всё, сервер включился! Обязательно оставьте это окошко открытым, не закрывайте его!**

---

### Шаг 4: Запуск САЙТА (То, что вы видите)
1. Вернитесь в главную папку `bus-ssys`.
2. Зайдите внутрь папки `frontend`.
3. Снова напишите `cmd` в адресной строке папки (как в прошлом шаге) и нажмите Enter. Откроется еще одно черное окошко.
4. Введите там команду:
   ```bash
   npm install
   ```
   *Опять подождите загрузку.*
5. Затем запустите сайт командой:
   ```bash
   npm run dev
   ```
6. В конце окошко напишет вам что-то вроде `Local: http://localhost:5174/`. 

### Шаг 5: Пользуемся!
Просто откройте ваш обычный браузер (Google Chrome, Яндекс, Safari), напишите в адресной строке **http://localhost:5174** и нажмите Enter.

**ГОТОВО! ВЫ ХАКЕР! 😎** 
Система видеонаблюдения, искусственного интеллекта и управления маршрутами Алматы запустится прямо на вашем экране!
